## Install GLFW-GLEW-OpenGL-GLM for Python

  **Python 3.x**
  * pip3 install pyopengl
  * pip3 install glfw
  * pip3 install glm
  * pip3 install numpy
  * pip3 install matplotlib
  
  **Python 2.x**
  * pip install pyopengl
  * pip install glfw
  * pip install glm
  * pip install numpy
  * pip install matplotlib
  
## Compile 

 **python3 test.py**
